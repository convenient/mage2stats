This website aims to work as an addendum to the standard analytics available on GitHub. The following graphs and tables allow us to track the flow of Issues and Pull Requests over time, and to easily gauge any improvements in GitHub activity.

The data for this report will be manually generated at the beginning of each month. If I'm running late feel free to contact me or crunch the numbers yourself and submit a PR.

Use the sliders below the graphs to roll up/drill down on the data.
